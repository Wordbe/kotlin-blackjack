- [x] : 이름을 가진 플레이어를 생성한다.
- [x] : 덱을 생성하고 카드를 뽑는다.

- [x] : 딜러가 처음에 받은 2장 합계가 16이하면 hit 해야하고, 17이상이면 stay 해야 한다.
- [ ] : 딜러가 21을 초과하면 그 시점까지 남아있는 플레이어들은 전부 승리한다.
- [ ] : 게임 완료 후 플레이어별 승패를 출력한다.
